# Final Project Front End

Front-end application built with React and Vite.

Live demo: [final-project-front-end-deplouy.vercel.app](https://final-project-front-end-deplouy.vercel.app)

## About

This project is a front-end web app created as a final project.  
It uses React for the UI, React Router for navigation, and Vite for development and build tooling.

The app includes pages and routes for:

- Home
- Login
- Signup
- Collection
- Decks
- Deck Details
- Edit Deck

It also includes protected routes and an authentication context.

## Tech Stack

- React
- Vite
- React Router DOM
- Axios
- CSS

## Getting Started

### 1. Clone the repository

bash
git clone https://github.com/Tchioleca/Final-project-front-end-deplouy.git
cd Final-project-front-end-deplouy
2. Install dependencies

If you are working inside the front-end folder:

cd front-end
npm install
3. Start the development server
npm run dev
4. Build for production
npm run build
5. Preview the production build
npm run preview
